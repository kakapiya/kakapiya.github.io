ES6基础

let const var辨析

let和const在声明变量时有块级作用域的概念

由此会发现，若在for循环中，声明i的方式不同，最后的结果也会不同。

若用let则为每次循环，i都会生成一个作用域；而用var，则始终改变的是该for循环下的全局变量i。

let 变量

常用于循环。

const 常量

引用模块时声明为常量。

  1. const常量声明的同时必须赋值，否则报错。

  2. const常量不可改的是引用值。
